# 🛰️ TITAN COMMAND v21.0
### OpenEnv Emergency Response Simulation

This is a real-world strategic simulation for AI agents and human commanders to manage urban crisis triage.

## 🚀 Quick Start
1. The **Backend** (FastAPI) runs on port 8000.
2. The **Frontend** (Streamlit) runs on port 7860.
3. Use `test_agent.py` to run a baseline autonomous mission.
